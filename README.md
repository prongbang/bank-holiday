# Bank Holiday

### Install

```
go get github.com/prongbang/bank-holiday
```

### How to use

#### Get Financial Holiday to List

```go
package main

import (
	bank_holiday "github.com/prongbang/bank-holiday"
)

func main() {
   utility := bank_holiday.New()
   holidayMap := utility.GetFinancialHolidayList("2022")
}
```

```go

	bank_holiday "github.com/prongbang/bank-holiday"
)

func main() {
   utility := bank_holiday.New()

}
```
